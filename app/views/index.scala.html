Neoflix

Movie Recommender


About

Neoflix is an example app using the Neovigator visualization. It uses the Movie Lens dataset and the work of Marko A. Rodriguez as shown on his blog entry A Graph-Based Movie Recommender Engine with some tweaks as shown on A Graph-Based Movie Recommender Website. This is a Scala & Play 2.0 port of Neoflix.

Get started

Start using the browser by just typing in the id of an existing node. You can see the available nodes at this Neo4j server , try loading Node 1 to get started.

Quirks

There may be some unexpected behavior. Performance may drop significantly as the number of nodes increases.
